商用厨房设计方案

2020-05-10

商用厨房设计方案

1. 整体设计规范

  1. 面积划分

    • 规范面积划分100人约30m2，100以上每加1人加0.3m2，1000人以上超过部分每加1人加0.2m2。

      • 建议厨房占餐饮总面积的20%，不能低于10%。

    • 粗加工占比10%以上，主厨房占比50%以上，备餐间占比15%以上，消洗间占比15%以上。

    • 厨房主过道至少为1.5米，通道门至少1米，工作区过道至少1米，厨师区过道至少0.9米。

  2. 主要设备概算

    1. 小炒炉一个炉头 约50人
      • 部队，学校等统一类约100人。
    2. 大锅灶 约100人
      • 统一类约100人。
    3. 蒸柜，蒸箱
      • 单门蒸柜约200300人主食或100150人蒸菜。三层蒸柜约50100人主食或3050人蒸菜。
    4. 其它加热设备200人以下中餐厅不宜超过2台/种。
    5. 烤箱以两到三层为宜。

  3. 行业标准

    1. 脏净分离（出品路线与收残路线不交叉）
    2. 生熟分离（出品路线与进料路线不交叉）
    3. 冷热

  4. 尽量缩短输送流程

  5. 了解厨房的既定菜式，最大就餐人数，预计厨房设计档次。

2. 厨房区域划分

2.1 主要区域

  1. 储存区

    1. 更衣间
      • 更衣柜，水池
    2. 收货区
      • 平板车，电子货称
    3. 仓库
      • 主食仓库
        • 米面架，
      • 副食仓库
        • 平板货架
      • 冷藏库
        • 平板货架
      • 冷冻库
      • 平板货架

  2. 加工区

    1. 蔬菜加工间

      • 大单星水池，货架，多功能切菜机，土豆脱皮机。

    2. 鱼类加工间

      • 杀鱼台，货架

    3. 肉类加工间

      • 单星带平台，绞切肉机，货架

      • 靠近主厨房

  3. 主厨房

    1. 切配区

      • 双层平板工作台，双门平台雪柜，四（六）门冰箱，水池，台面立架

    2. 打荷区

      • 双通工作台，双通暖碟工作台

    3. 烹饪区

      • 炒炉，大锅灶，炉拼台，海鲜蒸柜，煲仔炉，矮汤炉，蒸饭车。

      • 厨房与餐厅之间采用双门双道。

  4. 洗消区

    1. 洗碗间

      • 收残台，洗碗机，水池，平板工作台，开水器。

      • 应靠近餐厅，厨房，并力求与餐厅在同一平面，避免与厨房其它功能区之间出现交叉流线。

    2. 保洁间

      • 消毒柜，碗柜

      • 靠近洗碗间

  5. 备餐区

    • 单通工作台，双门宴会保温车

    • 应处于餐厅，厨房低过渡地带

2.2 按需求配备区域

  1. 面点间
    1. 面点制作
      • 木案工作台，面粉车，搅拌机，和面机，压面机，烤箱，醒发箱，饼盘车，电饼档，冰箱
    2. 面点蒸煮
      • 蒸包炉，肠粉炉，粉面炉，小炒炉，煲仔炉，冰箱，吊天花蒸笼架
  2. 烧腊间
    • 烤鸭炉，烤猪炉，低汤炉。
      1. 腌制间
        • 双层工作台，水池，冰箱
      2. 风干间
        • 烧腊架
  3. 凉菜间
    1. 预进间
      • 水池
    2. 双门平台雪柜，单通工作台，制冰机，冰箱，水池
  4. 明档
    1. 点心明档
    2. 烧腊明档
    3. 凉菜明档
    4. 刺身明档
      1. 杀鱼台
  5. 售卖区
    1. （四，五，六格保温售饭柜），保饭车，保汤车，

5. 关于排烟

  1. 以炉头计算：每个炉头取20002500风量m3/h计算，蒸柜2000，煲仔炉，低汤炉取10001500

  2. 以距离计算：一般每米烟罩取2000~2500m3/h计算

  3. 烟管风速一般取10~15m/s

  4. 风管截面积估算：A = Q/U*3600 面积=风量/风速x3600

  5. 风管截面尺寸估算：（bxh)=A 例：设A=0.36m2 主管高0.6 （0.6xh）=0.36 h=0.6

  6. 截面积>0.07m2 选矩形风s管 < 0.07选圆形风管

  7. 烟罩吸口管（分烟管）一般设250x300，最大不超过300x300（单位mm）

  8. 风量和功率关系表（估算）

1. java概述

1. java1995年由sum公司研发

2. 2006年sum将大部分java作为免费开源软件发布

3. 2009年由甲骨文公司收购

4. 2019年jdk8之后版本开始收费（jdk8现今企业开发常用版本）

2. java特性

1.java是可移植的。可以一次编译多平台运行。原理是代码编译后会生成class文件（字节码）。由java虚拟机jvm翻译成各自平台的机器码，不同的平台都可以安装各自的jvm。

2.java是一门面向对象编程语言

3.比较适合大型网站的后台开发例如淘宝，京东等

4.java是Android的主要开发语言（有小部分开发者使用谷歌推荐的Android官方开发语言kotlin）

5.java也可以开发windows程序（如我的世界）

3. java安装

甲骨文官网download页面下载，下载完之后无脑next即可。

安装完毕之后开始配置环境变量：点击计算机——属性——高级——环境变量设置。环境变量分两个：系统环境变量以及用户环境变量。我个人比较喜欢在用户环境变量里面配置。当然一般教程推荐在系统环境变量里面配置，这个看个人爱好，个人电脑的话基本没有影响，如果是公司电脑建议在系统用户变量里面配置。

配置变量：

1.JAVA_HOME （全部大写）：填写jdk的安装路径

2.Path （首字母大写） ： 填写jdk的bin路径（高级配置法是%JAVA_HOME%\bin配置好之后开始测试是否配置成功。

win+r建打开run 输入cmd进入DOS命令窗口依次输入java，javac,出现配置成功的文本即表示配置成功 最后输入java -version查看安装的jdk版本。

4. 下载开发环境

目前市面上主流的java开发环境只有eclipse以及idea现在建议直接使用更加智能的idea学习，安装方法为百度IDE官网找到download页面选择想要下载的版本即可，安装的话自行百度对应版本的破解教程。

进行第一个程序的编写

####java是一个面向对象的编程语言也就是类与对象的关系。首先新建一个类为你的类起一个名字。

编写完毕之后点击运行 或者shift+alt+f10快捷运行。无报错运行成功则完成了你的第一个程序编写。

1. 一个源文件可以有多个类但只能有一个公有类（public)

源文件名必须跟公有类名一致（区分大小写）
